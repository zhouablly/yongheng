# YǑNG HÉNG 永恒 — 设计规范 (DESIGN)

> **版本**: v1.0  
> **视觉概念**: Light Archive / 光的档案馆

---

## 1. 视觉哲学

### 1.1 核心意象
- **过曝胶片**: 边缘微微泛白，像老照片被阳光晒过的痕迹
- **晨雾白**: 清晨海边的薄雾，朦胧又温柔
- **海面蓝**: 退潮后的海水色，平静深邃
- **褪色纸张**: 旧信纸的质感，时间的痕迹
- **微光桃色**: 黄昏最后一抹光，温暖克制

### 1.2 审美铁律
- ❌ 绝对不用黑金奢华风
- ❌ 绝对不用粉色恋爱风
- ❌ 绝对不用蓝紫 AI SaaS 风
- ✅ 必须呈现：胶片感、留白、慢动画、真实情绪

## 2. 色彩系统

### 2.1 主色板

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `--porcelain` | `#F7F3EA` | 背景主色，瓷白 |
| `--milk` | `#FFFBF3` | 卡片背景，奶白 |
| `--mist-blue` | `#D7E4EA` | 薄雾蓝，辅助色 |
| `--sea-glass` | `#AFC8C6` | 海玻璃，强调色 |
| `--faded-peach` | `#E8B8A2` | 褪桃色，温暖强调 |
| `--sun-wash` | `#F2D6A2` | 日光黄，点缀 |
| `--paper-brown` | `#B69B7D` | 纸棕，文字次要 |
| `--soft-ink` | `#202124` | 软墨，主文字 |
| `--ash-gray` | `#8E8A82` | 灰褐，辅助文字 |
| `--line` | `rgba(32,33,36,0.12)` | 分割线 |

### 2.2 使用原则
- 大面积使用 porcelain 和 milk
- 用 mist-blue 和 sea-glass 做层次
- faded-peach 只用于强调和点缀
- 文字以 soft-ink 为主，ash-gray 为辅

## 5. 动效规范

### 5.1 核心原则
- **慢**: 所有动画都要慢，像呼吸一样
- **轻**: 动效幅度要小，不打扰
- **柔**: 缓动函数要柔和，不能生硬

### 5.2 动画参数
- 基础时长: 0.8s - 1.2s
- 缓动函数: `cubic-bezier(0.16, 1, 0.3, 1)`
- 延迟策略: 错落 100ms - 200ms
- 滚动驱动: GSAP ScrollTrigger scrub

### 5.3 禁用动效
- 快速闪烁
- 大幅度弹跳
- 3D 翻转
- 粒子爆炸

## 6. 组件风格

### 6.1 按钮
- 圆角: 2px（几乎直角，微微圆润）
- 边框: 1px solid var(--line)
- 背景: 透明或 milk
- 悬停: 微妙的底色变化
- 按压: 轻微下沉

### 6.2 卡片
- 背景: var(--milk)
- 边框: 1px solid var(--line)
- 阴影: 几乎不可见的软阴影
- 圆角: 4px

### 6.3 输入框
- 底部边框风格
- 无多余装饰
- 聚焦时柔和的光晕

---

## 7. 胶片质感

### 7.1 颗粒效果
- 全局胶片颗粒叠加层
- 透明度: 3-5%
- SVG noise 实现
- pointer-events: none

### 7.2 过曝效果
- 图片边缘微微提亮
- 用 radial gradient 模拟
- 微妙不张扬

### 7.3 褪色处理
- 图片降低一点饱和度
- 微微偏暖调
- 像被时间冲刷过
