## 1. 产品概述

### 1.1 产品定位
**永恒 · YǑNG HÉNG** 是一个以"光的档案馆"为视觉概念的爱情纪念 SaaS 平台。为相爱的人建造一座专属的数字档案馆，用光影、文字和时间收藏每一刻温柔。

### 1.2 核心价值主张
- **仪式感**: 创建专属空间如同开启一段神圣的仪式
- **轻量珍藏**: 不需要繁复操作，轻轻收藏就好
- **时间质感**: 像旧胶片一样，越久越有味道
- **私密永恒**: 只属于两个人的秘密花园

### 1.3 目标用户
- 18-35 岁的情侣/夫妻
- 注重生活仪式感
- 喜欢用影像记录生活
- 对视觉审美有较高要求

---

## 2. 核心功能

### 2.1 永恒空间
每个情侣拥有一个专属的"永恒空间"，包含：
- **封面**: 两人名字 + 纪念日 + 自定义封面图
- **时光计数器**: 在一起多少天的温柔计数
- **照片墙**: 按时间线排列的珍贵瞬间
- **时间轴**: 重要里程碑事件
- **情书**: 写给对方的私密信件
- **时光胶囊**: 写给未来的信，定时开启

### 2.2 创建仪式
四步创建专属空间：
1. 写下你们的名字
2. 选择开始的那一天
3. 上传第一张照片
4. 写下第一句悄悄话

### 2.3 视频叙事
首页通过滚动控制的视频讲述一段爱情故事：
- 从教室的那束光开始
- 到海边起舞的剪影结束
- 6 个章节，层层递进

---

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载 < 2s
- 视频预加载策略
- 滚动动画 60fps

### 3.2 视觉要求
- 严格遵循 Light Archive 设计语言
- 所有动效舒缓、克制
- 留白充足，呼吸感强

### 3.3 技术栈
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Framer Motion
- React Bits + Cult UI

---

## 4. MVP 范围界定

### 包含
- 首页展示（含 ScrollFilm）
- 创建空间流程（localStorage）
- 示例情侣空间（/space/demo）
- 基础响应式适配

### 不包含
- 真实支付系统
- 后端数据库
- AI 功能
- 邮件通知
- 自定义域名
- 复杂权限管理
