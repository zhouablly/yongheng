## 3. 内容层级

### 3.1 首页内容优先级
1. **Hero**: 品牌印象 + 核心价值
2. **ScrollFilm**: 情感共鸣 (灵魂区域)
3. **产品揭橥**: 一句话说清是什么
4. **功能档案**: 具体能做什么
5. **Demo 预览**: 眼见为实
6. **创建仪式**: 如何开始
7. **定价**: 需要多少钱
8. **收束**: 温柔收尾

### 3.2 空间内容优先级
1. **封面**: 一眼认出是"我们"
2. **时光计数器**: 时间的重量
3. **照片墙**: 最直观的回忆
4. **时间轴**: 故事的脉络
5. **情书**: 深度情感连接
6. **时光胶囊**: 面向未来的仪式

## 4. 用户状态

### 4.1 访客状态 (未登录/未创建)
- 可浏览首页全部内容
- 可查看 demo 空间
- 可走创建流程
- 数据存在 localStorage

### 4.2 有空间状态 (已创建)
- 可访问自己的空间
- 可编辑空间内容
- 可管理设置
- 数据存在 localStorage

## 5. 关键页面定义

### 5.1 首页 (/)
- **目的**: 品牌传达 + 情感共鸣 + 转化引导
- **核心元素**: ScrollFilm 滚动电影
- **CTA**: 创建我的永恒空间

### 5.2 创建空间 (/new)
- **目的**: 引导用户完成空间创建
- **核心元素**: 四步向导
- **结束**: 跳转至新创建的空间

### 5.3 空间页 (/space/[slug])
- **目的**: 展示情侣的专属档案
- **核心元素**: 照片墙 + 时间轴 + 情书
- **氛围**: 私密、温柔、专属

### 5.4 定价页 (/pricing)
- **目的**: 展示方案，促进升级
- **核心元素**: 三档定价对比
- **MVP**: 仅展示，不可购买
