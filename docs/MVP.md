# YǑNG HÉNG 永恒 — MVP 范围文档

> **版本**: v1.0  
> **阶段**: MVP / Demo

---

## 1. MVP 目标

构建一个可运行、可演示、视觉惊艳的爱情纪念 SaaS Demo，核心展示：
- "光的档案馆"视觉概念
- ScrollFilm 滚动视频叙事
- 创建空间的仪式感
- 示例情侣空间的美感

**不追求**：真实后端、完整功能、商业闭环。

---

## 2. 功能范围

### 2.1 ✅ MVP 包含

#### 首页 (Landing)
- Hero 区域：品牌宣告 + 进入 CTA
- ScrollFilm：6 章滚动视频叙事（核心灵魂）
- 产品揭橥：三句话说清永恒是什么
- 档案功能：照片墙 / 时间轴 / 情书 / 时光胶囊
- Demo 预览：示例空间入口
- 创建仪式：四步创建流程引导
- 定价方案：三档展示（仅展示，不可支付）
- 收束结语：温柔的结束语

#### 创建空间 (/new)
- 四步向导式流程
  - 第一步：你们的名字
  - 第二步：开始的日子
  - 第三步：第一张照片
  - 第四步：一句悄悄话
- localStorage 持久化
- 完成后跳转至专属空间

#### 示例空间 (/space/demo)
- 封面：姓名 + 纪念日 + 天数计数
- 照片墙：瀑布流/网格布局
- 时间轴：重要事件节点
- 情书：一封示例信件
- 时光胶囊：写给未来的信

#### 辅助页面
- /pricing 定价页
- /login 登录页（静态展示）
- /dashboard 仪表盘（静态展示）

### 2.2 ❌ MVP 不包含

#### 后端相关
- 真实服务器/数据库
- 用户注册/登录系统
- 图片上传存储（仅 URL 或 base64）
- 数据同步与备份

#### 商业相关
- 真实支付系统
- 订阅管理
- 发票/收据
- 优惠券系统

#### AI 相关
- AI 生成照片
- AI 写情书
- AI 视频生成
- 智能回忆推荐

#### 高级功能
- 自定义域名
- 多人协作
- 复杂权限管理
- 评论/点赞社交功能
- 打印/实物周边

---

## 3. 技术范围

### 3.1 ✅ 包含
- Next.js 14 App Router
- TypeScript 类型安全
- Tailwind CSS 样式
- GSAP + ScrollTrigger 滚动动画
- Framer Motion 微交互
- Lucide React 图标
- React Bits 组件库
- Cult UI 组件库
- localStorage 数据持久化

### 3.2 ❌ 不包含
- Three.js / WebGL 重型 3D
- 大型粒子系统
- 服务端渲染复杂逻辑
- 真实 API 集成
- 测试覆盖（只保证手工可用）

---

## 4. 交付标准

### 4.1 可运行
- `npm run dev` 正常启动
- `npm run build` 构建通过
- 主要路由可正常访问

### 4.2 可演示
- 首页滚动流畅
- 视频播放控制准确
- 创建流程可走完
- 示例空间可浏览

### 4.3 可看
- 视觉达到 Light Archive 标准
- 动效舒缓不突兀
- 响应式基本可用
